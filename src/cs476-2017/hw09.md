ITP
===


```{ pipe="tee nat-au.maude" }
fmod NAT-AU is
    sort Nat* .
    op 0   :           -> Nat* [ctor] .
    op 1   :           -> Nat* [ctor] .
    op _+_ : Nat* Nat* -> Nat* [ctor assoc id: 0] .
endfm

```

First, we spin up maude and load the IPT tool:

```{ pipe="tee prove-commutativity.maude" }
load nat-au.maude
load itp-tool.maude
loop init-itp .

(goal commutative : NAT-AU |- A{ X:Nat* ; Y:Nat* } ((X + Y) = (Y + X)) . )
(ind on Y:Nat* .)

--- case case
(auto .)

--- --- inductive step
--- (cns  .)        --- Fix X
--- (auto .)

--- induction on X
(ind on X:Nat* .)

(auto .)
(auto .)
(auto .)

(cns  .)
(imp  .)
(cns .)
```
